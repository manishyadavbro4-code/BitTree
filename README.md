# 🌳 BitTree

BitTree is a modern link-in-bio web application that allows users to organize and share all their important links from a single page. Perfect for creators, developers, freelancers, and businesses who want to showcase their online presence in one place.

## 🚀 Features

- Create a personalized profile page
- Add and manage multiple links
- Custom profile image and bio
- Responsive design for all devices
- Clean and modern user interface
- Easy link sharing

## 🛠️ Tech Stack

- Next.js
- React.js
- Tailwind CSS
- MongoDB
- Node.js
